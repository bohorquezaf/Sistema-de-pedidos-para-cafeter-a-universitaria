# Lista de Requerimientos funcionales

----

| Requerimiento | Descripción |
|---------------|-------------|
| Registro de productos | Permite agregar, editar o eliminar productos. |
| Visualización del menú | Muestra lista actualizada de productos. |
| Registro de pedidos | Permite seleccionar productos y generar orden. |
| Actualización de estado | Cambiar estado a *En preparación*, *Listo*, *Entregado*. |
| Reportes | Generar informes de venta y productos. |
| Autenticación | Validar acceso según tipo de usuario. |
| Historial de pedidos | Mostrar pedidos anteriores con información detallada. |
